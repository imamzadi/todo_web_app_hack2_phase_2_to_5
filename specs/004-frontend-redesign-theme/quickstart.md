# Quickstart: Frontend Redesign with New Theme

This feature is a UI redesign of the frontend application. To see the changes, you need to run the frontend application.

## Running the Frontend

1.  Navigate to the `frontend` directory:
    ```bash
    cd frontend
    ```

2.  Install the dependencies:
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  Open your browser and navigate to `http://localhost:3000` to see the redesigned application.
